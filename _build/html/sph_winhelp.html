SphinxでWinHelp形式(.chm)ファイルを作成¶

Windowsのヘルプファイルを作れる模様。

HTML Help Workshop のインストール¶

前もってインストールしておきます。

Sphinx のインストール¶

前もってインストールしておきます。

Sphinxドキュメントルートに¶

以下を置く。

ソースコード(_cv_hhc.py)¶

# -*- coding: utf-8 -*-
#
# 文字変換するよ(index)
#  "_build/htmlhelp/remedyhelpdoc.hhc"が生成されていること

import os
import sys
import re

import conf

# .hhk
cvpath = '_build\\htmlhelp\\'+conf.htmlhelp_basename + '.hhc'

# def
def foo(match):
    n = int(match.groups(0)[0])
    try:
        s = unichr(n).encode('mbcs')
    except:
        s= "?"
    return s


#
p= re.compile(r'\&\#(\d*)\;')

fi = open(cvpath,'r')

buf = []
while 1:
    line = fi.readline()

    if not line:
        break

    buf.append(p.sub(foo,line))

fi.close()

#

fo = open(cvpath,'w')

for x in buf:
    fo.write(x)

fo.close()

ソースコード(_cv_hhp.py)¶

# -*- coding: utf-8 -*-
#
# 文字変換するよ(設定ファイル)
#

import os
import sys
import conf

# .hhp
cvpath = '_build\\htmlhelp\\'+conf.htmlhelp_basename + '.hhp'

fi = open(cvpath,'r')

buf = []
while 1:
    line = fi.readline()

    if not line:
        break

    if 'Default Font=' in line:
        buf.append('Default Font=MS UI Gothic,8,128')
    elif 'Language=' in line:
        buf.append('Language=0x411')
    else:
        buf.append(line)

fi.close()

#

fo = open(cvpath,'w')

for x in buf:
    fo.write(x)

fo.close()

ソースコード(_cv_iso10646h.py)¶

# -*- coding: utf-8 -*-
#
# htmlファイルの変換を行なうよ
#  "_build/htmlhelp"にhtmlファイルがあること。

import os
import sys
import re

# .hhc
cvpath = r'_build\htmlhelp'


# def
def foo(match):
    n = int(match.groups(0)[0])
    try:
        s = unichr(n).encode('mbcs')
    except:
        s= "?"
    return s



#
p= re.compile(r'\&\#(\d*)\;')

lines= os.popen("dir /B /A-D "+cvpath+"\\*.html","r")
sys.stdout.flush()

while 1:

    filename = lines.readline()
    filename = filename.rstrip()

    if not filename: break

    fi = open(cvpath+"\\"+filename,'r')

    buf = []
    
    while 1:
        line = fi.readline()

        if not line:
            break

        line = line.replace('content="text/html; charset=iso8859_1"','content="text/html; charset=shift-jis"')
        buf.append(p.sub(foo,line))

    fi.close()
    
    fo = open(cvpath+"\\"+filename,'w')
    
    for x in buf:
        fo.write(x)
    
    
    fo.close()

call .\make.bat htmlhelp

.chmファイルを生成する¶

ドキュメントルートの相対パス、”_build/htmlhelp”に移動します。 HTML Help Workshopで .hhp ファイルを開きます。

コンパイルします。

動かないときは¶

動かないときは

  • conf.py の htmlhelp_basename はありますか？
  • conf.pycを削除して再度実行。
  • _build/htmlhelp ディレクトリはありますか？
  • _build/htmlhelp ディレクトリにきちんとファイルが生成されていますか?
  • 生成されたhtmlファイルの中身に “&#xxxx;” と表記されていませんか?

その他¶

なんかよくわかりません。
