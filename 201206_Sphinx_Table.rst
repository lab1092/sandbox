=================================
Sphinxのテーブルについて(複数行)
=================================



Q.
==

.. code-block:: none

   == ==
   a  b 
   == ==
   c
   d  e
   == ==


期待:

+-----+---+
| a   | b |
+=====+===+
|     |   |
| | c |   |
| | d | e |
+-----+---+



結果:

== ==
a  b 
== ==
c
d  e
== ==


   

A-1. "複雑な書き方"を使う
==========================

.. code-block:: none

   +-----+---+
   | a   | b |
   +=====+===+
   |     |   |
   | | c |   |
   | | d | e |
   +-----+---+


+-----+---+
| a   | b |
+=====+===+
|     |   |
| | c |   |
| | d | e |
+-----+---+

A-2. "list-tableを使う"
=======================

.. code-block:: none 

   .. list-table::
      :header-rows: 1
   
      * - a
        - b
      * - | c
          | d
        - e


.. list-table::
   :header-rows: 1

   * - a
     - b
   * - | c
       | d
     - e
